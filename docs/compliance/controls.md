# Compliance Controls Mapping (Initial)

## SOC 2

- Access control and least privilege for command APIs.
- Change management through versioned policy files.
- Security event logging for every autonomous action.

## GDPR/CCPA

- Minimize personal data in telemetry payloads.
- Data retention limits and deletion workflow.
- Purpose limitation: mission operation and safety only.

## ISO-aligned Safety Engineering

- Hazard analysis and risk assessment artifacts.
- Traceability from hazard -> requirement -> test evidence.
- Verification gates before expanding operational design domain.
